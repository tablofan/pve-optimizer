# PvE Optimizer

A standalone browser tool for optimizing **oasis farming** in Travian. Designed for **x3 speed** servers running **Travian T4.6**.

## Goal

Help decide which oases to send troops on — ranking/optimizing oasis raid targets for resource efficiency.

## File Structure

| File | Purpose |
|------|---------|
| `index.html` | Main application — UI, parsing, calculation logic, styling |
| `DOCS.md` | This documentation file |

## Technical Notes

- Single-page app, no build step — open `index.html` directly in a browser
- Dark Travian-themed UI matching the `trade-route-calculator` sibling project
- All state in-memory; nothing persisted
